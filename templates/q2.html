Question 2

Survey
  • This question is about HTML forms and how to process them in Flask.
  • The Python part of your answer has to be written in the app.py file, in the proper places, as indicated in the TODO comments.
  • You will need to create a HTML template (file templates/survey.html), extending the exam template.

Steps

  1. Create a FlaskForm for a survey in the file app.py. Put your code where indicated in the TODO comments.
    1. The first field in your form should be an email input field (the person's email address who's filling the survey)
    2. The second field should be a checkbox. If checked, the person's email address should be hidden in the results. Otherwise, the person's email address is recorded in the results.
    3. The third field should be a color field (the person's favorite color).
    4. Then there should be a radio button group, corresponding to the choices a, b, c, d of a multiple-choice question (MCQ). You have to write your own question with 4 choices.
  2. Add the proper validators to your form. Each field should be mandatory except for the checkbox field. Make sure the email field contains a valid email address.
  3. Create an endpoint for your survey form in the file app.py. Put your code where indicated in the TODO comments.
    1. The route should be /survey, and it should accept both theÂ GET and POST methods. It should follow the POST/REDIRECT/GET pattern
    2. Method GET: displays the form
    3. Method POST: save the survey results in a CSV file
    4. Redirect to /survey/results on success (this endpoint has to be implemented in question 3)
